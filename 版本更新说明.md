# ycIDE 版本更新说明

## ycIDE - 开发测试版 v0.0.3-beta.53(2026-05-04)

---

### 新增功能

- [新增]打开项目对话框支持直接选择 `.e/.ec` 文件导入，支持读取工程头、密码输入与同名目录冲突处理。
- [新增]表格模式代码显示优化：流程命令 `判断开始` 在表格模式下显示为 `判断`，文本模式保持原始命令不变。

### 问题修复

- [修复]修复 `.e/.ec` 导入读取原生工程段长度偏移错误导致的部分项目 `Magic` 报错。
- [修复]修复 `.e/.ec` 导入只生成“导入说明”或空子程序占位的问题，导入后可生成真实 `EYC/EGV/EDT/ELL/ERC` 内容。
- [修复]修复核心命令号映射偏差：补齐 `Lib0` 回退映射并覆盖 `181/182/203` 兼容点，减少 `_Lib0CmdXXX` 占位显示。
- [修复]修复非核心支持库命令映射缺失：补充多支持库命令 ID 映射，减少 `_LibXCmdY` 回退。
- [修复]修复特殊结构下（`判断开始` 与后续 `判断` 分离）内侧线缺失、断段与贯穿不完整问题，补齐上一行起始横线及跨行内侧竖线连通。
- [修复]修复“第 7 行下三角箭头残留”系列问题，统一屏蔽重叠分支下不应显示的下箭头。
- [修复]修复内侧线“上半段显示规则”异常：仅对“从上方延续”的行显示上半段，避免首行误绘。
- [修复]修复 `如果真` 结束箭头在隐藏结束行后的显示异常：补回向下三角并去除多余横向虚线。
- [修复]修复嵌套 `如果真` 场景中结束箭头锚定错层与漏显问题，改为按同深度独立锚定与绘制。
- [修复]修复手工输入 `.如果` 回车未补齐可见 `否则/如果结束` 的问题，并修复文本/表格模式可见性不一致。
- [修复]修复表格编辑器偶发“临时回退文本模式”与编辑中空白闪烁问题。
- [修复]修复并清理 VS Code Problems 面板剩余告警（ARIA、inline style、类型收窄、不兼容 CSS）。

### 优化调整

- [优化]流程线与流程命令解析进一步收敛为“命令驱动”路径，去除旧 marker 兼容分支的生效逻辑，降低流程缩进与连线异常来源。
- [优化]表格模式性能系统性优化：可视区 block/行级裁切、滚动节流、缓存复用、minimap 阈值收紧，降低大文件卡顿。
- [优化]将语法诊断与表格模型构建迁移到 Worker，减少主线程阻塞与首开空白时间。
- [优化]补强“判断自动展开”诊断日志，新增关键判定上下文，便于定位第 5/7 行结构未命中原因。
- [优化]调整首屏渲染策略：先渲染轻量前置行，再异步切换完整模型，改善大文件打开体验。

## ycIDE - 开发测试版 v0.0.3.52(2026-04-28)

- [新增]AI 助手新增 Plan 模式与 Agent 模式，并支持项目级文件工具调用和终端工具调用闭环。

- [修复]修复手工输入 `.如果` 后按回车未自动补齐 `否则/如果结束` 可见流程结构的问题。
- [修复]修复流程自动补齐出的 `否则/如果结束` 在文本模式可见但表格模式被旧样式隐藏的问题。
- [修复]修复 Windows 启动后窗口左上角图标仍显示默认四方块的问题。
- [修复]修复自绘标题栏中“文件”菜单左侧仍显示四方块占位图的问题，改为显示程序 logo。

### 优化调整

- [优化]程序图标统一切换为 `ycidelogo.svg` 并接入 Windows/macOS/Linux 打包资源。
- [优化]优化表格模式流程线计算，减少大文件中分支、结束和相邻流程连接判断的重复扫描。
- [优化]AI 助手内置模型升级与扩展（DeepSeek-V4-Pro、DeepSeek V4-Flash、DeepSeek V3.2）。
- [优化]终端交互体验优化（按需启动、输入区一体化、中文输出解码稳定性）。

## ycIDE - 开发测试版 v0.0.0.30 (2026-03-15)

### 新增功能
- [新增]属性面板现在支持直接编辑属性值，点击属性值即可进入编辑模式，输入新值后按回车确认修改


### 问题修复
- [修复]流程控制语句（如 如果、判断）在某些情况下会破坏代码结构的问题
- [修复]编译后底部控件被截断问题，缺少标题栏高度计算


### 优化调整
- [优化]移除工具栏右侧冗余的"支持库管理"按钮（已在菜单"工具"中提供）
- [优化]可视化设计器中窗口标题现在改为跟win11默认样式一样，并且将选中窗口整体改为选中客户区
