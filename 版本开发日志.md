## 2026-04-24（终端输出稳定性修复）

- 优化：修复 `cmd.exe` 在中文 Windows 下终端输出乱码问题，新增终端输出解码逻辑，优先 UTF-8，检测到替换符时自动回退 GBK（CP936）解码。
- 优化：终端系统日志（已启动/当前目录）改为去重写入，避免重复触发启动流程时出现连续重复提示。
- 涉及文件：`src/main/index.ts`

## 2026-04-24（终端快捷键与内联输入交互修复）

- 修复：终端支持 `Ctrl+C` 中断当前命令，主进程新增 `terminal:interrupt`，中断后自动重建 shell 会话。
- 优化：终端输入改为控制台内联输入，移除独立输入框；光标在控制台末尾闪烁，输入位置与终端最后一行保持一致。
- 优化：终端区域聚焦后可直接键盘输入，支持 `Enter` 执行、`Backspace` 删除和粘贴文本。
- 涉及文件：`src/main/index.ts`、`src/preload/index.ts`、`src/renderer/src/App.tsx`、`src/renderer/src/components/OutputPanel/OutputPanel.tsx`、`src/renderer/src/components/OutputPanel/OutputPanel.css`

## 2026-04-24（Agent 工具结果流式回传修复）

- 修复：Agent 执行工具步骤时，工具日志改为实时回调到 AI 助手消息流，避免“执行完但窗口没有结果”的空白体验。
- 修复：`terminal_send` 工具在执行后自动抓取新增终端输出并回传到 Agent 对话，确保如 `node --version` 这类命令结果可直接显示在 AI 助手窗口。
- 优化：保留最终汇总日志兜底，避免极端情况下流式日志遗漏。
- 涉及文件：`src/renderer/src/App.tsx`、`src/renderer/src/components/AIAssistantPanel/AIAssistantPanel.tsx`

## 2026-04-24（AI 助手交互与 Agent 意图路由优化）

- 新增：AI 助手输入区增加“停止”按钮，可立即中止当前会话的流式显示与后续结果写入（聊天 / 计划 / 编辑 / Agent）。
- 修复：AI 聊天记录区域恢复文本多选与复制能力。
- 优化：Agent 模式移除固定占位输出，减少每次执行都出现的重复提示行。
- 修复：当用户目标包含网络连通性语义（如“检查网络连接”）时，Agent 优先走终端 `ping` 诊断流程，而不是先扫描项目源码文件。
- 涉及文件：`src/renderer/src/App.tsx`、`src/renderer/src/components/AIAssistantPanel/AIAssistantPanel.tsx`、`src/renderer/src/components/AIAssistantPanel/AIAssistantPanel.css`

## 2026-04-24（DeepSeek Agent Function Calling 升级）

- 新增：AI 通信层支持 `tools` / `tool_choice` / `tool_calls` 协议，兼容 DeepSeek 的函数调用 Agent 模式。

- 优化：保留项目级文件与终端工具能力，并继续对网络连通性任务优先使用终端命令诊断。
